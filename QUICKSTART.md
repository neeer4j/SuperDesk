# SuperDesk

A modern remote desktop access web application for Windows, featuring real-time screen sharing, bidirectional audio, file transfer, and secure peer-to-peer connections.

## Quick Start

```bash
# Clone the repository
git clone https://github.com/neeer4j/SuperDesk.git
cd SuperDesk

# Install all dependencies
npm run install:all

# Start development environment
npm run dev:server  # Terminal 1
npm run dev:client  # Terminal 2  
npm run dev:agent   # Terminal 3
```

## Access Points

- **Web Client**: http://localhost:3000
- **Server**: http://localhost:3001
- **Desktop Agent**: Electron application
